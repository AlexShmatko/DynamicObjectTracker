# Dynamic Object Tracker

## Installation

Install dependencies:

- Install CMake using Homebrew to manage the build process.

  ```bash
  brew install cmake
  ```

- Install OpenCV

  ```bash  
  brew install opencv
  ```
  
## Build

Create a build directory:

```bash
mkdir build && cd build
```

Run CMake and Make:

```bash
cmake .. 
make
```

## Usage

Execute the application:

```bash
./DynamicObjectTracker
```
